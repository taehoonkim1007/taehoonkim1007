## 개발자 김태훈

신규 MES 시스템 개발 단계에서 DB 스키마 설계를 기획하고 생산 프로세스 자동화를 운영하며, 재고 일치율을 90%에서 98%로 향상시킨 경험은 제게 요구사항을 분석하고 체계적인 시스템으로 설계하는 것의 가치를 체감하게 했습니다.

이를 바탕으로, AI 캐릭터 채팅 서비스에서 도메인에 맞는 적응형 대화 시스템을 설계하고, 팩트체크 서비스에서 멀티스텝 검증 파이프라인을 구축하며 '요구사항에 맞는 시스템 설계'를 실천해 왔습니다.

앞으로도 특정 기술에 얽매이지 않고, 요구사항에 최적화된 시스템을 설계하여 사용자와 비즈니스 모두에게 가치를 전달하는 서비스를 만들어가겠습니다.
